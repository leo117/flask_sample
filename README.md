### run the app with flask


Firstly initialize db

```
python init_db.py
```

on linux

````commandline
export FLASK_APP=app
export FLASK_ENV=development
flask run
````

on Windows

````commandline
set FLASK_APP=app
set FLASK_ENV=development
flask run
````
